# libaudio

Go wrappers for `ffmpeg` and `ffprobe`.

## fsutil

Exports file system functions that are commonly used with `ffmpeg` and `ffprobe` wrappers.

## Developing

### Testing

From within source path, run:

    go test -cover -v ./...
